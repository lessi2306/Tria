Tria - Version 11.5.8

Folgendes wurde korrigiert:

  • Fehler beim Speichern der Tria-Datei.
  • Fehler bei der Definition von Altersklassen.

Systemanforderungen

Tria benötigt ein Microsoft Windows Betriebssystem. Unterstützt werden Windows XP und spätere Versionen.

Für Import und Export von Microsoft Excel Dateien sollte Excel XP oder eine spätere Version auf dem Rechner installiert sein und für die Erstellung von Urkunden ist ein Textbearbeitungsprogramm mit Serienbrieffunktion, vorzugsweise Micosoft Word, erforderlich.

Für weitere Informationen siehe www.selten.de.
