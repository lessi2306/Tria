; Text Pop-ups für Tria
.topic IDH_Home
Für dieses Element ist kein Hilfethema vorhanden.

.topic IDH_Hilfe
Zeigt eine Beschreibung dieses Dialogfensters.

.topic IDH_Abbr
Schließt das Dialogfenster, ohne die vorgenommenen Änderungen zu speichern.

.topic IDH_Ok
Speichert die vorgenommenen Änderungen und schließt das Dialogfenster.

.topic IDH_Aend
Speichert die vorgenommenen Änderungen ohne das Dialogfenster zu schließen.

.topic IDH_Neu
Erstellt einen neuen Eintrag in der Liste

.topic IDH_Loesch
Löscht den markierten Eintrag aus der Liste. 

.topic IDH_Up
Verschiebt den Eintrag in der Liste um eine Position nach oben.

.topic IDH_Down
Verschiebt den Eintrag in der Liste um eine Position nach unten.

.topic IDH_VeranstName
Der Name der Veranstaltung erscheint in den Kopfzeilen der ausgedruckten Listen.

.topic IDH_VeranstArt
Wählen sie einen einmaligen Wettbewerb oder eine Serie von 2 bis 20 Wettbewerben, die an verschiedenen Orten und zu verschiedenen Terminen durchgeführt werden.

.topic IDH_OrtName
Zeigt den Namen des in der Liste markierten Austragungsortes.

.topic IDH_OrtList
Klicken Sie auf einen Ortsnamen, um diese im Feld "Ortsname" zu ändern.

.topic IDH_WkLst
Klicken Sie auf einen Wettkampf, um die zugeordneten Einstellungen in den übrigen Feldern an zu zeigen.

.topic IDH_WkName
Der Name des markierten Wettkampfes erscheint als Überschrift auf den ausgedruckten Listen.

.topic IDH_WkDatum
Das Datum des Wettkampfes erscheint in den Überschriften der ausgedruckten Listen. Zusätzlich wird aus diesem Datum und dem eingegebenen Geburtsjahr das Alter und die Altersklasse der Teilnehmer ermittelt.

.topic IDH_WkArt
Neben einem normalen Einzelwettkampf, bei dem eine Einzelwertung und optional eine Mannschaftswertung durchgeführt wird, können auch spezielle Mannschaftswettkämpfe gewählt werden, bei denen die Mannschaftswertung im Vordergrund steht.

Klicken Sie auf den Schalter "Hilfe" für weitere Details.

.topic IDH_WkAbschnZahl
Sie können eine Anzahl von 1 bis 8 Wettkampfdisziplinen definieren, die nacheinander durchgeführt werden. Beim Einzelwettkampf wird für jeden Abschnitt eine eigene Teilnehmerwertung durchgeführt.

.topic IDH_WkAbschnName
Wenn mindestens zwei Abschnitte definiert sind, erscheinen die Abschnittsnamen als Spaltenüberschrift auf den ausgedruckten Listen. Es kann eine Bezeichnung aus der Liste augewählt oder frei definiert werden.

.topic IDH_WkAbschnRunden
Für jeden Abschnitt kann eine beliebige Anzahl Runden definiert werden. Der Abschnitt wird nur gewertet, wenn für jede Runde eine Rundenzeit eingelesen wird.  

.topic IDH_WkStrecke
Bei Stundenrennen muss die Länge einer Runde in Metern definiert werden. Daraus wird die insgesamt zurückgelegte Strecke errechnet.   

.topic IDH_WkSondWrtg
Bei der Sonderwertung werden nur Teilnehmer berücksichtigt, die gesondert für diese Wertung angemeldet werden. Damit kann zusätzlich zur Gesamtwertung z.B. eine Stadt- oder Kreismeisterschaft gewertet werden

.topic IDH_WkTlnTxt
Das optionale Eingabefeld für Teilnehmerdaten kann hier mit einer beliebigen Bezeichnung versehen werden (z.B. zur Angabe von Nation, Bundesland oder T-Shirt-Größe). 
Diese Daten können in einer entsprechenden Spalte auf den Start- und Ergebnislisten für Teilnehmer ausgedruckt werden. Die relativ geringe Spaltenbreite erlaubt einen Text bestehend aus 3 Groß- oder 4 Kleinbuchstaben.

.topic IDH_WkStartgeld
Das hier eingetragene Startgeld wird bei der Neuanmeldung von Teilnehmern voreingestellt. Der Betrag kann im Teilnehmerdialog individuell angepasst werden.

.topic IDH_WkDisqTxt
Die hier eingetragene Bezeichnung wird für disqualifizierte Teilnehmer voreingestellt. Sie kann im Teilnehmerdialog individuell angepasst werden.

.topic IDH_WkMschZahl
Wenn keine Mannschaftswertung gewünscht wird, kann das hier eingestellt werden. Andernfalls wird für alle Teilnehmer, für die ein Mannschaftsname eingetragen wurde, eine Mannschaftswertung durchgeführt. Dabei werden alle Teilnehmer mit gleichem Mannschaftsnamen automatisch in einer oder mehreren Mannschaften zusammengefasst.

Klicken Sie auf den Schalter "Hilfe" für weitere Details.

.topic IDH_WkMschModus
Die Wertung der Mannschaften geschieht entweder durch Addition der Endzeiten oder Platzierungen der Teilnehmer. Ein zusätzlicher Modus ist für Veranstaltungen im Rahmen der DTU Schultour vorgesehen 

Klicken Sie auf den Schalter "Hilfe" für weitere Details.

.topic IDH_WkMschGroesse
Eine Mannschaft wird nur gewertet, wenn mindestens die hier für die Klasse eingestellte Anzahl von Mannschaftsteilnehmern das Ziel erreicht. Es können Mannschaftsgrößen von 2 bis 16 Teilnehmer definiert werden. 

.topic IDH_WkSchwDist
Hier kann die Anzahl der Bahnen, die geschwommen werden müssen, definiert werden (max. 30). Diese Angabe wird für die Erstellung einer Checkliste für Bahnenzähler verwendet. Diese Liste kann in der Ansicht "Checkliste Schwimmbahnen" ausgedruckt werden.

.topic IDH_WkSchwBahn
Wenn die Teilnehmer in Startbahnen eingeteilt werden sollen, kann hier die Anzahl der verfügbaren Schwimmbahnen definiert werden (max. 16). Jedem Teilnehmer muss dann eine spezifische Startbahn zugeordnet werden. Diese wird in der Startliste in einer zusätzlichen Spalte dargestellt.

.topic IDH_WkSerName
Der Wettkampfnamen wird hier einheitlich für alle Austragungsorte definiert. Dieser Name wird nur für die verschiedenen Wettkampf-Auswahllisten im Programm verwendet. Auf den ausgedruckten Listen erscheint der unten definierte, orts-spezifische Wettkampfname.

.topic IDH_WkKuerzel
Bei Liga-Veranstaltungen kann zusätzlich zum Name ein Kürzel für den Wettkampf festgelegt werden, das in der Liga-Spalte auf der "Ergebnisliste Teilnehmer" verwendet wird.

.topic IDH_SGrpLst
Klicken Sie auf eine Startgruppe, um die zugeordneten Einstellungen in den übrigen Feldern an zu zeigen.

.topic IDH_SGrpName
Sie können für jede Startgruppe einen unterschiedlichen Namen festlegen. Die Startgruppen werden nach Name sortiert in der Liste dargestellt.

.topic IDH_SGrpWettk
Die markierte Startgruppe wird einem der definierten Wettkämpfen zugeordnet. Für jeden Wettkampf muss mindestens eine Startgruppe definiert werden.

.topic IDH_SGrpSnr
Aus dem definierten Nummernbereich können die Startnummern der in dieser Startgruppe eingeteilten Teilnehmern ausgewählt werden.

.topic IDH_SGrpTlnMax
Die maximale Anzahl der Teilnehmer, die in der markierten Startgruppe eingeteilt werden können, wird aus dem Startnummernbereich errechnet. Sie dient lediglich zur Information.

.topic IDH_SGrpTlnIst
Die Anzahl der in der markierten Startgruppe eingeteilten Teilnehmer dient lediglich zur Information.

.topic IDH_SGrpAbs1
Die Startzeit des ersten Abschnitts gilt für alle Teilnehmer, die in dieser Startgruppe eingeteilt werden.
In der Regel wird der erste Abschnitt als Massenstart durchgeführt. Alternativ können ein Jagdstart oder Einzelstart definiert werden. Beim Jagdstart starten die Teilnehmer nacheinander mit einem gleichen, unten festgelegten Zeitabstand. Beim Einzelstart wird die Startzeit für jeden Teilnehmer einzeln eingelesen.

Klicken Sie auf den Schalter "Hilfe" für weitere Details.
 
.topic IDH_SGrpAbs2
In der Regel werden die weiteren Abschnitten ohne Unterbrechung, unmittelbar in Anschluss am vorangegangenen Abschnitt, gestartet. Alternativ kann eine Pause und dazu eine neue Startzeit festgelegt werden. Nach einer solchen Unterbrechung kann der Start entweder als Massenstart, oder als Jagdstart definiert werden. Die Zeitabstände beim Jagdstart entsprechen den Zeitabständen am Ende des vorigen Abschnitts.

Klicken Sie auf den Schalter "Hilfe" für weitere Details.

.topic IDH_TlnName
Name und Vorname müssen für jeden Teilnehmer eingetragen werden. Teilnehmer mit gleichem Namen und Vornamen sind erlaubt, wenn sie an verschiedenen Wettkämpfen teilnehmen oder im gleichen Wettkampf, wenn sie verschiedenen Mannschaften angehören.

.topic IDH_TlnSex
Das Geschlecht muss für jeden Teilnehmer eingestellt werden. Es kann sowohl das männliche, das weibliche als auch kein Geschlecht unter dem Menüpunkt "Extras/Optionen" als Voreinstellung gewählt werden.

.topic IDH_TlnJg
Das Geburtsjahr des Teilnehmers kann hier eingetragen werden. Wenn dieses fehlt, wird der Teilnehmer bei den Altersklassenwertungen nicht berücksichtigt.

.topic IDH_TlnAlter
Das Alter des Teilnehmers wird aus dem Wettkampf-Datum und dem Geburtsjahr errechnet. Diese Angabe dient nur zur Information.

.topic IDH_TlnAk
Die Einteilung in Altersklasse und Sonderklasse wird aus dem Wettkampfdatum und dem Geburtsjahr des Teilnehmers ermittelt. Dabei werden nur die für den Wettkampf definierten Altersklassen und ggf. Sonderklassen benutzt. Diese Angabe dient nur zur Information.

.topic IDH_TlnMsch
Der hier eingetragene Vereins- oder Ortsname wird in allen Teilnehmerlisten in einer eigenen Spalte dargestellt. Wird ein Name eingetragen und die Teilnahme an der Mannschaftswertung im Bereich "Optionen" definiert (Voreinstellung), dann wird dieser Name auch als Mannschaftsname für die Mannschaftswertung verwendet.

Der Vereins- oder Ortsname kann entweder direkt eingetragen oder aus einer Liste aller bereits eingegebenen Namen ausgewählt werden. Es ist zu empfehlen, nach Möglichkeit die Auswahlliste zu verwenden um unterschiedliche Schreibweisen der Mannschaftsnamen und damit eine inkorrekte Mannschaftswertung zu vermeiden.

.topic IDH_TlnWettk
Durch auswahl aus der Liste, wird der Teilnehmer einem der definierten Wettkämpfen zugeordnet.
 
.topic IDH_TlnNext
Speichert die eingegebenen Daten und erzeugt ein neues Anmelde-Fenster.

.topic IDH_TlnNav
Mit diesen Navigations-Schaltern können andere Teilnehmer aus der Liste ausgewählt werden, ohne das Dialogfenster zu verlassen.

.topic IDH_SMldLst
Klicken Sie auf einen Vereinsmelder, um die zugeordneten Einstellungen in den übrigen Feldern und die gemeldeten Teilnehmer in der Tabelle an zu zeigen.

.topic IDH_SMldSort
Die Vereinsmelder können nach Name oder nach Verein sortiert in der Liste dargestellt werden.

.topic IDH_SMldName
Name und Vorname müssen eingetragen werden. Sie können für die Erstellung von Adressetiketten verwendet werden.

.topic IDH_SMldMsch
Der Mannschaftsname kann eingetragen oder aus einer Liste aller bereits eingegebenen Mannschaften ausgewählt werden. Er wird als Vorgabe bei den gemeldeten Teilnehmern eingetragen.

.topic IDH_SMldLand
Das Land wird als Vorgabe bei den gemeldeten Teilnehmern eingetragen.

.topic IDH_SMldAdr
Die Angabe von Adresse und E-Mail ist optional. Die Adresse kann für die Erstellung von Adressetiketten verwendet werden.

.topic IDH_SMldDelAll
Alle Sammelmelder in der Liste werden gelöscht.

.topic IDH_TlnSMld
Dem Teilnehmer kann einen Vereinsmelder zugeordnet werden. In der Ansicht "Anmeldung-Verein" ist dieser fest vorgegeben. 

.topic IDH_TlnAdr
Die Angabe von Adresse und E-Mail ist optional. Die Adresse kann für die Erstellung von Adressetiketten verwendet werden.

.topic IDH_TlnSchw
Es kann eine Meldezeit, z.B. eine voraussichtliche Schwimmzeit eingetragen werden. Diese kann benutzt werden, um bei einem Start im Schwimmbad die Teilnehmer entsprechend ihrer Schwimmleistung in Startgruppen und Bahnen einzuteilen.

.topic IDH_TlnLand
Dieses optionales Eingabefeld ist nur vorhanden, wenn es im Wettkampf-Dialog auf der Registerkarte "Teilnehmer" freigegeben wurde. Dort wird auch die Bezeichnung des Feldes definiert. Die Daten können in einer zusätzlichen Spalte auf Start- und Ergebnislisten für Teilnehmer dargestellt werden. Die Spaltenbreite ist begrenzt und erlaubt einen Text bestehend aus 3 Groß- oder 4 Kleinbuchstaben. Mögliche Anwendungen sind z.B. die Angabe von Nationalität oder Bundesland oder die Größe eines T-Shirts.

.topic IDH_TlnStGeld
Die Angabe eines Startgeldes is optional. Der Wert wird nur in der Anmelde-Ansicht angezeigt. Die Liste kann dabei auch nach dieser Wert sortiert werden.

.topic IDH_TlnRfid
Wenn die Zeitnahme mittels RFID-Code als Option gewählt wurde, kann hier ein RFID-Code eingetragen werden.  

.topic IDH_TlnKomm
Das Feld "Kommentar" steht für beliebige Einträge zur freien Verfügung. Der Inhalt dieses Feldes wird beim Export in Excel- und Textdateien und in den Seriendruckdateien für Etiketten und Urkunden ausgegeben, hat in Tria aber keine weitere Funktion.

Klicken Sie auf den Schalter "Hilfe" für weitere Details.

.topic IDH_TlnMschWrtg
Teilnahme an der Mannschaftswertung ist nur möglich, wenn das Feld "Verein/Ort" ausgefüllt wurde und der Teilnehmer nicht außer Konkurrenz startet und an der Serienwertung teilnimmt (bei Serienveranstaltungen).

.topic IDH_TlnMixMsch
Wenn eine Mannschaftswertung eingestellt ist, kann der Teilnehmer für eine Mixed-Mannschaftswertung angemeldet werden. Er wird dann bei den nach Geschlecht oder Altersklassen gewerteten Mannschaften nicht berücksichtigt.
 
.topic IDH_TlnSondWrtg
Teilnahme an der Sonderwertung kann nur eingetragen werden, wenn für den Wettkampf eine Sonderwertung definiert ist und der Teilnehmer nicht außer Konkurrenz startet.

.topic IDH_TlnAusKonk
Es ist möglich, Teilnehmer außer Konkurrenz teilnehmen zu lassen. Diese Teilnehmer erscheinen in der Ergebnisliste an einer ihrer Gesamtzeit entsprechenden Position, sie werden jedoch bei der Berechnung der Platzierungen nicht berücksichtigt. Anstelle der Platzierung wird das Kürzel "a.K." eingetragen.

.topic IDH_TlnSerWrtg
Teilnahme an der Serienwertung kann nur eingetragen werden, wenn die Veranstaltung als Serienveranstaltung definiert ist und der Teilnehmer nicht außer Konkurrenz startet. Teilnehmer, die nicht an der Serienwertung teilnehmen, sind auch von der Mannschaftswertung ausgeschlossen.

.topic IDH_TlnUrk
Nur wenn diese Option eingestellt ist, wird ein Teilnehmer bei der Erstellung der Seriendruckdatei für Urkunden berücksichtigt.

.topic IDH_TlnStfName1
Der Hauptteilnehmer ist als Starter für den 1. Wettkampfabschnitt vorgegeben.

.topic IDH_TlnStfName2
Hier werden die übrigen Staffelteilnehmer definiert. Der Name muss eingetragen werden, der Vorname ist optional.  

.topic IDH_TlnSGrp
Klicken Sie auf eine Startgruppe, um den Teilnehmer in diese Startgruppe einzuteilen. Es stehen nur die für den gewählten Wettkampf definierten Startgruppen zur Auswahl. Zur besseren Übersicht wird die Anzahl der bereits eingeteilten Teilnehmer für jede Startgruppe in der Liste angezeigt.

.topic IDH_TlnSBhn
Wenn für diesen Wettkampf Startbahnen definiert wurden, können Sie dem Teilnehmer eine Startbahn aus der Liste zuordnen. Zur besseren Übersicht wird die Anzahl der bereits eingeteilten Teilnehmer für jede Startbanh in der Liste angezeigt. Wenn für die Anzahl der Startbahnen die Zahl null definiert wurde, dann wird diese Liste gesperrt.

.topic IDH_TlnSnr
Die Startnummer kann entweder manuell in das Feld eingetragen werden, oder aus der Liste der verfügbaren Startnummern in der gewählten Startgruppe ausgewählt werden.

.topic IDH_TlnStZt1
Dieses Feld zeigt die Startzeit des ersten Abschnitts. Wenn für die gewählte Startgruppe einen Massenstart definiert wurde (Normalfall), entspricht sie die für die Startgruppe definierten Startzeit. Beim Jagdstart wird sie aus Startzeit und Zeitabstand abgeleitet. Beim Einzelstart wird die Startzeit bei der Zeitnahme gemeinsam für alle Teilnehmer aus einer Datei eingelesen. Sie kann in diesem Fall hier auch manuell eingetragen und modifiziert werden, diese Möglichkeit ist aber nur für Ausnahmefälle vorgesehen.
 
.topic IDH_TlnStZt2
Die Startzeit des Abschnitts wird aus der gestoppten Uhrzeit des vorangegangenen Abschnitts und dem Startmodus dieses Abschnitts abgeleitet. Bei einem Start ohne Unterbrechung ist sie identisch mit der Stoppzeit des vorangegangenen Abschnitts.

.topic IDH_TlnUhrZt
Die für diesen Abschnitt gestoppte Uhrzeit wird bei der Zeitnahme gemeinsam für alle Teilnehmer aus einer Datei eingelesen. Sie kann hier auch manuell eingetragen und modifiziert werden, diese Möglichkeit ist aber nur für Ausnahmefälle vorgesehen.
 
Es ist jede Uhrzeit erlaubt, auch 00:00:00[,00] ist eine gültige Uhrzeit. Eine nicht erfasste Zeit wird durch ein leeres Feld gekennzeichnet. Dies kann manuell erreicht werden durch Löschen der Uhrzeit mit der Leertaste oder der Entf-Taste.

.topic IDH_TlnZtGtSchr
Die errechnete Endzeit kann auch eine Zeitgutschrift beinhalten. Diese wird auf der Registerkarte "Gutschrift/Strafe" definiert.

.topic IDH_TlnStrafZt
Die errechnete Endzeit kann auch eine Strafzeit beinhalten. Diese wird auf der Registerkarte "Gutschrift/Strafe" definiert.

.topic IDH_TlnAbsZt
Die für diesen Abschnitt gebrauchte Zeit entspricht der Differenz der oben gezeigten Start- und Stoppzeit.

.topic IDH_TlnEndZt
Die Endzeit des Teilnehmers entspricht der Summe von Abschnittszeiten und Strafzeit, vermindert mit der Zeitgutschrift.   

.topic IDH_TlnAbsRnd
Wenn für den Abschnitt mehr als eine Runde definiert wurde, werden die Rundenzeiten in einem zusätzlichen Fenster angezeigt. In dem benachbarten Feld kann die ausgewählte Rundenzeit bearbeitet werden.

.topic IDH_TlnAbsRndLst
Wenn für den Abschnitt mehr als eine Runde definiert wurde, werden die Rundenzeiten in diesem Fenster angezeigt. In dem darüber liegenden Feld kann die ausgewählte Rundenzeit bearbeitet werden.

.topic IDH_TlnGesRng

.topic IDH_TlnGesAbs

.topic IDH_TlnSexRng

.topic IDH_TlnSexAbs

.topic IDH_TlnAkRng

.topic IDH_TlnAkAbs
Zeigt die Platzierung in der Altersklassenwertung über diesen Abschnitt.

.topic IDH_TlnZusAbs
Zeigt die Platzierung in der Sonderklassenwertung über diesen Abschnitt.

.topic IDH_TlnSerTag
Zeigt die Tagesplatzierung im aktuellen Austragungsort, die für die Serienwertung verwendet wird.

.topic IDH_TlnSerPkt
Zeigt die Gesamtpunktzahl in der Serienwertung über alle Austragungsorte.

.topic IDH_TlnSerRng
Zeigt die Platzierung in der Serienwertung über alle Austragungsorte.

.topic IDH_TlnStrZt
Bei einer Zeitstrafe wird die eingetragene Zeit zu der Gesamtzeit hinzugezählt. Es kann auch die Zeit 00:00 als Strafe eingetragen werden, wenn die Zeitstrafe schon während des Wettkampfes "abgesessen" wurde. In der Ergebnisliste wird eine Zeitstrafe, auch wenn diese gleich null ist, durch ein *-Zeichen nach der Endzeit gekennzeichnet.

.topic IDH_TlnDisq
Disqualifizierte Teilnehmer können, wenn gewünscht, in die Ergebnisliste aufgenommen werden. Dies wird über das Auswahlfeld "Status" auf der Ansichtsleiste bestimmt. Wenn sie gelistet werden, dann werden disqualifizierte Teilnehmer am Ende der Ergebnisliste angehängt. Statt der Platzierung wird die hier definierte Bezeichnung (Vorgabe "disq") eingetragen.
Bei einer Disqualifikation kann zusätzlich eine Begründung eingetragen werden. Dies hat lediglich einen informativen Zweck. 

.topic IDH_TlnGutschr

.topic IDH_TlnRestStr
Bei Stundenrennen kann hier die Reststrecke, die nach dem letzten Zieldurchlauf zurückgelegt wurde, in Metern eingetragen werden. Diese wird bei der Berechnung der Gesamtstrecke berücksichtigt.
